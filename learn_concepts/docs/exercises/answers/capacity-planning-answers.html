Exercise 1: Forecast Capacity

Question: Forecast capacity needs for next 6 months. What's the method?

Answer

Forecasting Method: 1. Historical data: Analyze historical growth 2. Trend analysis: Identify growth trends 3. Business projections: Use business growth projections 4. Calculate: Calculate future capacity needs

Answer: Use historical data, trend analysis, business projections to forecast capacity.

Exercise 2: Plan Capacity

Question: Plan capacity additions for 2× growth. What's the strategy?

Answer

Capacity Plan: 1. Measure current: Measure current capacity 2. Calculate needs: Calculate capacity for 2× growth 3. Plan additions: Plan capacity additions (compute, storage, network) 4. Execute: Execute plan before needed

Answer: Measure current, calculate 2× needs, plan additions, execute proactively.

Exercise 3: Handle Capacity Shortage

Question: Your system is running out of capacity. How do you handle it?

Answer

Immediate Actions: 1. Scale up: Scale up resources immediately 2. Optimize: Optimize current usage 3. Shed load: Shed non-critical load 4. Plan: Plan for capacity additions

Answer: Scale up immediately, optimize usage, shed load, plan capacity additions.
